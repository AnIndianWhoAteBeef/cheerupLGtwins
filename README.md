# ⚾ KBO Fan Platform

KBO 팬들을 위한 올인원 웹 플랫폼입니다.
경기 정보 확인부터 응원 참여까지, 팬 경험을 더 재미있게 만드는 것을 목표로 합니다.

---

## 🚀 주요 기능 (Features)

### ❤️ 1. 응원하기 시스템

* 당일 팀별 응원 버튼 클릭 수 집계
* 상대 팀과 실시간 비교
* 시즌 누적 응원 수 랭킹 제공 (KBO 순위표 형태)
* 날짜별 응원 기록 저장
* 매일 기록된 응원 수를 토대로 그래프로 만들어 동향 파악 기능
* 하루에 인당 3표씩 팀 상관 없이 응원 가능
* 더 커지면 1군에 1경기 이상 뛴 선수들 모아서 선수 인기투표(하루 인당 3표)

---

### 📊 2. 실시간 경기 상황

* 금일 경기 진행 상황 표시
* 이닝, 점수, 상태 (진행 중 / 종료 등)

---

### 🧾 3. 경기 결과 (히스토리)

* 이전 경기 스코어 제공
* 날짜별 경기 결과 조회

---

### 🎟️ 4. 구단별 티켓 예매 링크

* 각 구단 공식 티켓 예매 페이지 연결

---

### 📅 5. KBO 경기 일정

* 다가오는 경기 일정 제공
* 날짜별 경기 리스트

---

### 📰 6. KBO 뉴스 & 기사

* 매일 업데이트되는 KBO 관련 기사 제공

---

## 🛠️ 기술 스택 (Tech Stack)

* Frontend: React / Next.js
* Backend: Node.js / Express
* Database: MongoDB (또는 Firebase)
* API: KBO 데이터 제공 API (추후 연동)
